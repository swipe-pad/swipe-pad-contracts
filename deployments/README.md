# Deployments

This folder stores contract deployment outputs created by Foundry.

Generated by:

```bash
forge script script/Deploy.s.sol --rpc-url <RPC_URL> --broadcast
```

Outputs:
- `deployments/<chainId>.json` (written by `Deploy.s.sol`)

Notes:
- Do not commit private keys.
- Commit deployment JSON for staging/prod so the frontend can be synced.
